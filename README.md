A Python application that dynamically generates visual backgrounds based on real-world conditions such as weather, time of day, and season.

This project is currently an active prototype using temporary and placeholder art assets. 

## Features
Dynamic weather backgrounds
Time-of-day sky transitions
Seasonal overlays
Holiday overlays
Lightweight wallpaper updates
Integrates weather and time data
Dynamically layers visual elements (sky, fog, clouds, etc.)
Combines PNG assets to create a final rendered image
Tech
Python
Image processing (PIL or similar)

## Setup
Install Python
Get OpenWeather API key
Set environment variable
Create config.json
Run scheduler

## Project Goals
Build a creative automation tool combining real-world data with visual generation logic.
